Font Data Analysis

Objective
This program computes the length of a given input string in pixels and the average distance between the centers of characters in that string.
If given a pair of input strings the program will compare the string lengths and character distances.
Input (There are two types of input)
Simple input for measurement only
  1. There is a input character text set (called the ChrSet)
  2. A font with the following parameters given: the font name, the font size, the letter-spacing and the word-spacing.
Paired Input for comparison purposes
  1. One input character string as above
  2. Two fonts with varying values of the parameters above for the purpos of comparing differences resulting from changing font parameters.
Output
Measurement
  1. SLPx — This is the string length in pixels.
  2. AveChar — This is the average distance between the centers of characters in pixels AveChar = SLPx / (the number of characters in the input string).
Comparison
szRatio — the ratio of the SLPx values for comparison sets.
